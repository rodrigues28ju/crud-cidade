Crud-Cidade
Criamos uma aplicação para uso pessoal que consegue armazenar dados com as funcionalidades do GET, PUT, POST e DELETE.

Rodando localmente
Clone o projeto

  git clone
Entre no diretório do projeto

  cd crud-cidade
Instale as dependências

  npm  install ou yarn install
npm run mock
Iniciar o servidor

apêndice
